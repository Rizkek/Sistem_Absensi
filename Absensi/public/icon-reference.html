Cara Penggunaan di Filament Resource

// Di StudentResource.php
protected static ?string $navigationIcon = 'heroicon-o-users';

// Ganti dengan Material Symbol (setelah Filament support custom icons):
// Untuk sementara, icon di navigation masih pakai Heroicons
// Tapi di custom view/blade, bisa pakai Material Symbols bebas

// Contoh di Blade file:
<span class="material-symbols-outlined">school</span>

Catatan: Filament default menggunakan Heroicons untuk navigation. Material Symbols bisa digunakan di custom blade views, widgets, dan komponen custom.
